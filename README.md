# UEPluginDevTemplate

Unreal Engine 插件开发模板，用于快速搭建插件开发与测试环境。

## 核心功能

- 标准空白 UE 项目，可直接放置插件到 `Plugins/` 下开发测试
- 预配置 `.gitignore`、LICENSE、文档结构
- 支持多插件并行开发
- 内置文档同步工具（MZSH-UEPlugins 组织专用）

## 适用场景

- 插件原型开发、框架验证、性能测试
- 多插件并行开发测试
- FAB / GitHub 发布前的打包准备环境
